# SOS Casa Express — Landing Page

Landing page estática baseada nas referências visuais fornecidas, focada em serviços para imobiliárias: limpeza profissional, pintura e pequenos reparos.

## Como usar

- Requisitos: já vem com Node/npm (checado neste ambiente).
- Para rodar em localhost:
  1. Na raiz do projeto, execute: `npm run dev`
  2. Abra `http://localhost:5173`
- Sem npm: abra diretamente `index.html` no navegador (sem auto‑reload).
- Estilos em `styles/main.css` e JS em `scripts/main.js`.

## Notas

- As imagens utilizam links do Unsplash para ilustração.
- O formulário é apenas demonstrativo e não envia dados para backend.
- Cores, textos e seções seguem o estilo das referências: destaque em laranja, cartões com sombra, seções com painéis escuros e carrosséis simples.

## Vídeo no Hero

- O arquivo `hero-vid.mp4` é usado como plano de fundo da seção Hero (autoplay, loop, muted, playsinline).
- Fallback para quem usa “reduzir animações”: imagem estática (Unsplash).
